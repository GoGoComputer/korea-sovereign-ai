# 🇰🇷 한국 소버린 AI — 사용 가이드

**🌐 Language**: **한국어 (현재)** · [English](en/3-usage.md)

> 컴퓨터에 설치된 한국 AI 모델(EXAONE, A.X, Solar)을 **인터넷 없이** 내 맥북에서 바로 쓰는 방법.
> 개발자가 아니어도 따라 할 수 있도록 정리했습니다.

---

## 📌 한국 소버린 AI가 뭔가요?

**소버린 AI(Sovereign AI)** = "주권 AI". 우리나라 기업·연구소가 직접 만든 AI 모델로,
- 데이터가 해외로 나가지 않고
- 우리말을 제일 잘 이해하고
- 정부 "독자 AI 파운데이션 모델(독파모)" 사업의 핵심 후보입니다.

이 노트북에는 한국 대표 3종이 깔려 있어요:

| 모델 | 만든 곳 | 한 줄 소개 |
|---|---|---|
| **EXAONE 4.0 1.2B** | LG AI연구원 | 작고 빠름. 가벼운 질문/요약에 최적 |
| **A.X 4.0 Light** | SK텔레콤 | 한국어가 가장 매끄러움. 일상 대화·업무 메일 |
| **Solar Pro 22B** | Upstage | 가장 똑똑함. 복잡한 추론·코딩·번역 |

> 💡 **모두 인터넷 없이 동작합니다.** 질문 내용은 외부로 전송되지 않아요.

## 🚀 1분 안에 시작하기

터미널(Terminal 앱)을 열고 아래를 그대로 입력하세요:

```bash
cd ~/DEV/llmDev/korea-ai
./launcher.sh
```

그러면 이런 메뉴가 뜹니다:

```
===============================
 🇰🇷 Korean Sovereign AI Launcher
===============================
 (메모리 깨끗함)
 /exaone      EXAONE 1.2B (Ollama)        ~0.8GB
 /exaone-mlx  EXAONE 1.2B (MLX 4bit)      ~0.7GB ★ 가장 빠름
 /ax          SKT A.X 4.0 Light (MLX)     ~4.5GB ★ 한국어 강세
 /solar       Solar Pro 22B (Ollama)      ~13GB
 /unload      메모리 강제 정리
 /ps          현재 로드 모델 보기
 /help        도움말
 /quit        종료
-------------------------------
>
```

`>` 옆에 원하는 명령을 입력하고 엔터.

**예시:**
- `/ax` 입력 → A.X 4.0 한국어 채팅 시작
- `1` 또는 `/exaone` → EXAONE 시작
- `/quit` → 런처 종료

## 💬 대화하는 방법

모델이 시작되면 그냥 한국어로 질문하세요.

```
> /ax
▶ SKT A.X 4.0 Light (MLX 4bit) — ~4.5GB, 한국어 최강
  종료: Ctrl+D

>>> 회사 워크숍 일정 안내 메일 초안 써줘
[모델이 답변]

>>> 더 짧게 줄여줘
[모델이 답변]
```

### 채팅 끝내고 메뉴로 돌아가기

| 모델 종류 | 끝내는 법 |
|---|---|
| `/exaone`, `/solar` (Ollama) | 채팅창에 `/bye` 입력하고 엔터 |
| `/exaone-mlx`, `/ax` (MLX) | 키보드에서 `Ctrl + D` |

끝내면 자동으로 메뉴로 돌아옵니다. 거기서 다른 모델을 선택하면, **앞 모델은 알아서 메모리에서 빠집니다** (걱정 안 해도 됨).

## 🎯 어떤 모델을 골라야 하나요?

### 상황별 추천

**📩 한국어 메일 / 문서 / 일상 질문 → `/ax`**
- 가장 자연스러운 한국어 표현
- "공손하게 정중한 메일", "친구한테 카톡처럼" 같은 한국식 톤 잘 맞춤
- 메모리 ~4.5GB

**⚡ 빠른 답이 필요한 짧은 질문 → `/exaone-mlx`**
- 282 토큰/초, 거의 실시간으로 글자가 쏟아져 나옴
- 짧은 요약, 간단한 번역, 빠른 사실 확인
- 메모리 ~0.7GB (가장 가벼움)

**🧠 어려운 일 (코딩, 긴 추론, 전문 번역) → `/solar`**
- 22B로 가장 똑똑함
- 다만 메모리 13GB → 다른 무거운 앱(Chrome, Slack, VS Code)을 닫고 쓰는 게 좋아요
- 답변 속도는 느림 (대신 품질이 좋음)

**🤔 잘 모르겠으면 → `/ax`로 시작**
한국어 사용자 입장에서 가장 무난합니다.

## 🧹 메모리 관리

맥북 메모리가 24GB라 큰 모델을 띄우면 다른 앱이 느려질 수 있어요.

### 다 쓰고 메모리 깨끗하게 하기

런처 안에서:
```
> /unload
✓ 정리 완료
```

또는 터미널에서 바로:
```bash
./launcher.sh /unload
```

### 지금 뭐가 메모리에 떠 있는지 확인

```
> /ps
NAME                     SIZE     UNTIL
solar-pro:latest         15 GB    4 minutes from now
```

> 💡 모델을 바꿀 때(예: `/exaone` 쓰다가 `/solar` 선택)는 **자동으로** 앞 모델이 내려가니 신경 쓸 필요 없음. `/unload`는 "지금 당장 다 비워라" 강제 명령.

## 🆘 자주 묻는 질문

### Q. 인터넷 끊겨도 돼요?
A. **네, 완전히 오프라인으로 동작합니다.** 모델이 컴퓨터 안에 들어 있어요.

### Q. 내 질문/대화가 어디로 새지 않나요?
A. **새지 않습니다.** 외부 서버 호출 없이 맥북 내부에서만 처리돼요.

### Q. 모델이 답을 잘못 하면?
A. AI는 가끔 틀린 정보를 자신 있게 말합니다(환각). **중요한 결정에 쓰기 전엔 사실 확인 필수.**

### Q. 맥북이 너무 뜨겁고 느려져요
A. `/solar` (22GB) 같은 큰 모델 때문일 수 있어요. `/unload` 하거나, 평소엔 `/ax`나 `/exaone-mlx` 같은 가벼운 모델을 쓰세요.

### Q. 답변이 갑자기 영어로 나와요
A. 더 명확하게 "한국어로 답변해줘"를 질문 끝에 붙이거나, `/ax` (한국어 특화) 모델로 바꿔보세요.

### Q. 끄려면 어떻게 해요?
A. 메뉴에서 `/quit` 또는 `Ctrl + C`. (자동으로 메모리도 정리됨)

### Q. 다음에 또 쓰려면?
A. 터미널에서 다시 `cd ~/DEV/llmDev/korea-ai && ./launcher.sh` — 끝.

## 🛟 문제가 생기면

### "command not found" 같은 메시지가 나와요
경로가 맞는지 확인:
```bash
ls ~/DEV/llmDev/korea-ai/launcher.sh
```
파일이 보이면 OK. 안 보이면 개발자에게 문의.

### 메뉴에서 아무 것도 안 일어나요
`Ctrl + C` 눌러 빠져나오고 다시 시작:
```bash
./launcher.sh /unload   # 일단 메모리 정리
./launcher.sh           # 다시 시작
```

### 답변이 너무 느려요
- 큰 모델(`/solar`)을 쓰고 있다면 작은 모델(`/ax`, `/exaone-mlx`)로 바꿔보세요
- 다른 무거운 앱(Chrome 탭 많음, VS Code 등) 닫기

### 모델이 자꾸 헛소리해요
- 질문을 더 구체적으로: "메일 써줘" → "거래처에 미팅 일정 30분 미루는 양해 메일, 정중하게 5문장으로"
- 한국어 품질 우선이면 `/ax`, 추론 품질 우선이면 `/solar`
