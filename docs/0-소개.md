# 0. 이게 뭔가요? — 맥북 없는 분도 이해하는 가이드

**🌐 Language**: **한국어 (현재)** · [English](en/0-intro.md)

이 레포지토리를 처음 보는 분, 혹은 맥북이 없어도 "도대체 무슨 프로젝트지?" 가 궁금한 분을 위한 개념 설명서.

---

## 한 줄 요약

> **한국 회사들이 만든 AI 모델(EXAONE, A.X, Solar)을 인터넷 없이 내 맥북 안에서 돌리는 도구 모음.**

---

## 🤔 왜 만들었나?

### 문제 1. ChatGPT는 매번 미국 서버로 데이터를 보냅니다
- 회사 기밀, 개인정보, 의료기록 → 외부 서버 전송
- 인터넷 끊기면 사용 불가
- 사용량 한도, 구독 비용

### 문제 2. 한국 AI 모델은 한국어를 더 잘합니다
- 한국 정부 **"독자 AI 파운데이션 모델(독파모)"** 사업으로 LG·SKT·Upstage 등이 자체 모델 공개 (2025~)
- 한국어 전용 토크나이저 → 같은 비용으로 더 긴 한국어 처리
- 우리말 표현·존댓말·뉘앙스를 더 잘 이해

### 문제 3. 막상 쓰려니 너무 복잡합니다
- 모델마다 다운로드 방법 다름 (Ollama / MLX / Hugging Face)
- 메모리 24GB짜리 노트북에 13GB 모델 띄우면 다른 앱이 죽음
- 명령어 외우는 것도 귀찮음

→ **이 레포가 그 모든 걸 한 줄 명령으로 해결합니다.**

---

## 🇰🇷 "소버린 AI(Sovereign AI)"가 뭔가요?

| 일반 AI (ChatGPT 등) | 소버린 AI (이 레포) |
|---|---|
| 미국 회사가 미국 서버에서 운영 | 우리나라 회사가 만든 모델 |
| 데이터 해외 전송 | 내 컴퓨터 안에서만 처리 |
| 인터넷 필수 | 오프라인 가능 |
| 영어가 모국어 | **한국어가 모국어** |
| 외부 정책 영향 받음 | 우리나라 데이터 주권 유지 |

**"주권(Sovereign)"** = 데이터·기술·운영을 외부에 의존하지 않는 상태.

전세계 정부들이 자국 AI를 만들려는 이유:
- 🇰🇷 한국: "독파모" 사업 (LG, SKT, 네이버, Upstage, NCSOFT, KT 등)
- 🇫🇷 프랑스: Mistral
- 🇯🇵 일본: Sakana AI, ELYZA
- 🇨🇳 중국: Qwen, DeepSeek
- 🇦🇪 UAE: Falcon

## 📦 이 레포가 다루는 한국 모델

### LG AI연구원 — EXAONE 4.0
- **가벼움**: 1.2GB 정도. 노트북에 부담 없음
- **빠름**: 글자가 거의 실시간으로 쏟아져 나옴 (M5 Pro에서 282자/초)
- **용도**: 짧은 질문, 요약, 빠른 사실 확인
- **만든 곳**: LG그룹의 AI 연구 자회사 (2020년 설립)

### SK텔레콤 — A.X 4.0 Light
- **한국어 강자**: SKT가 자체 슈퍼컴퓨터(TITAN)로 학습
- **효율적**: GPT-4o 대비 한국어 토큰 33% 절약
- **상업 가능**: Apache 2.0 라이선스 (가장 개방적)
- **용도**: 일상 한국어, 메일, 문서 작성, 업무용

### Upstage — Solar Pro 22B
- **가장 똑똑함**: 22B 파라미터로 70B급 추론 성능
- **무거움**: 13GB → 24GB 노트북 한계까지 사용
- **MIT 라이선스**: 글로벌 노선의 한국 스타트업
- **용도**: 코딩, 복잡한 추론, 전문 번역

## 💻 어떤 컴퓨터가 필요한가?

| 사양 | 가능 모델 |
|---|---|
| **Mac M1~M5 Pro/Max, 16GB+** | EXAONE, A.X (가벼운 모델) |
| **Mac M1~M5 Pro/Max, 24GB+** | 위 + Solar (전체 라인업) ⭐ |
| **Mac M1~M5 Pro/Max, 64GB+** | 위 + 더 큰 미래 모델 (EXAONE 32B 등) |
| 인텔 Mac | ❌ MLX 미지원 (Ollama만 가능, 매우 느림) |
| Windows / Linux | ⚠️ 이 스크립트는 macOS 전용. 모델 자체는 다른 OS에서도 가능 (별도 설치) |

> 💡 **이 레포는 Apple Silicon (M 시리즈) 전용**으로 만들어졌습니다. 같은 모델을 Windows/Linux에서 쓰려면 [Ollama](https://ollama.com)를 직접 설치하고 모델을 따로 받으세요.

---

## ⚡ 어떻게 빠른가? — MLX 의 마법

Apple Silicon (M1~M5)은 CPU·GPU·메모리가 한 칩에 있는 **통합 메모리 구조**.
Apple이 만든 **MLX** 프레임워크는 이 구조를 100% 활용:

- 일반 노트북: GPU↔CPU 데이터 복사로 시간 낭비
- M5 Pro + MLX: 복사 없이 직접 접근 → **노트북에서 282자/초**

→ EXAONE 1.2B을 거의 ChatGPT 답변 속도와 비슷하게 돌릴 수 있어요.

---

## 🚫 이 레포가 하지 않는 것

- AI 모델을 **새로 만들지 않습니다** (LG/SKT/Upstage가 이미 만든 걸 가져다 쓰는 도구)
- ChatGPT 만큼 똑똑하지 않습니다 (1.2B~22B는 GPT-4o(~1T)보다 작음)
- 이미지 생성 / 음성 인식 / 동영상 분석 안 함 (텍스트 LLM 전용)
- 클라우드 배포 / API 서버 자동화 안 함 (개인 노트북용)
- Windows/Linux 미지원 (모델 자체는 가능, 이 스크립트는 macOS 전용)

## ❓ 자주 묻는 질문

### Q. 그래서 이게 ChatGPT 대체재인가요?
A. **부분 대체**입니다. 빠른 한국어 질문, 요약, 메일 작성 등 일상 업무는 충분.
복잡한 추론·최신 정보가 필요하면 여전히 ChatGPT/Claude가 낫습니다.

### Q. 인터넷 없이 정말 동작해요?
A. **네**, 한 번 다운로드만 받으면 그 다음부터 완전 오프라인.
비행기 안, 지하실, 보안망 환경에서도 사용 가능.

### Q. 회사 기밀 입력해도 되나요?
A. 외부로 안 나갑니다. 다만 **모델이 환각으로 잘못된 답을 줄 수 있으니** 중요한 결정엔 검증 필수.

### Q. 무료인가요?
A. **모델은 무료** (각 회사가 공개), **이 도구도 무료** (MIT 라이선스).
다만 모델별 라이선스(상업 가능 여부)는 다르니 확인 필요:
- A.X: Apache 2.0 (상업 OK)
- Solar: MIT (상업 OK)
- EXAONE: 연구용 무료, 상업은 별도 협의

### Q. 맥북이 없어도 이 코드 도움 되나요?
A. 직접 사용은 어렵지만 **참고 자료**로 유용:
- 한국 소버린 AI 모델 목록·특징 정리됨
- bash 스크립트 구조 (메모리 관리, 자동 설치)
- Ollama / MLX 사용법
