# 2. 처음 설치

새 맥북 또는 한 번도 안 깔아본 환경 기준.

## 사전 준비 (한 번만)

### 1. Ollama 설치
[ollama.com/download](https://ollama.com/download) 에서 macOS용 다운받아 설치 → Ollama.app 실행.

확인:
```bash
ollama --version
curl -s http://localhost:11434/api/tags    # 빈 줄이라도 응답 오면 OK
```

### 2. Python 3.9+ (macOS 기본 포함)
```bash
python3 --version              # 3.9+
```

### 3. (자동) 의존성
`install.sh` 가 알아서 설치하지만 수동으로 미리 깔려면:
```bash
python3 -m pip install --user mlx-lm huggingface_hub
```

`huggingface-cli` 가 `~/Library/Python/3.9/bin` 에 들어가니 PATH 확인:
```bash
echo 'export PATH="$HOME/Library/Python/3.9/bin:$PATH"' >> ~/.zshrc
source ~/.zshrc
which huggingface-cli
```

## 자동 설치 (권장)

```bash
cd ~/DEV/llmDev/korea-ai

./install.sh                # 대화형 (각 모델마다 확인)
./install.sh --yes          # 모두 자동
./install.sh --minimal      # EXAONE + A.X만 (~5GB, Solar 제외)
./install.sh --check        # 상태만 확인 (설치 안 함)
```

설치 단계 (멱등 — 이미 깔린 건 자동으로 skip):

| # | 단계 | 크기 | 시간 |
|---|---|---|---|
| 0 | 환경 점검 (Apple Silicon, RAM) | — | 1초 |
| 1 | PATH 등록 (`~/.zshrc`) | — | 1초 |
| 2 | Python 의존성 (mlx-lm, huggingface_hub) | — | 30초 |
| 3 | Ollama 동작 확인 | — | 1초 |
| 4 | EXAONE Ollama pull | 0.8GB | 1~3분 |
| 5 | EXAONE MLX 변환 | 0.7GB | 2~5분 |
| 6 | **A.X 4.0 BF16 다운로드 + 4bit 변환** | 14GB→4.5GB | 15~30분 |
| 7 | Solar Pro GGUF 다운로드 + 등록 | 13GB | 20~40분 |
| 8 | (옵션) 주간 자동 유지보수 cron 등록 | — | 1초 |

> ⏱ 전체 다운로드 합계 약 **28GB**, 시간 30~70분 (네트워크 속도 의존).
> --minimal 모드면 약 5GB / 5~10분.

## 수동 설치 (자동이 실패할 때)

### EXAONE Ollama
```bash
ollama pull hf.co/LGAI-EXAONE/EXAONE-4.0-1.2B-GGUF:Q4_K_M
```

### EXAONE MLX
```bash
python3 -m mlx_lm convert \
  --hf-path LGAI-EXAONE/EXAONE-4.0-1.2B \
  --mlx-path ~/models/exaone-4.0-1.2b-4bit-mlx -q
```

### A.X 4.0 Light MLX
```bash
python3 -m mlx_lm convert \
  --hf-path skt/A.X-4.0-Light \
  --mlx-path ~/models/ax-4.0-light-4bit-mlx -q
```

### Solar Pro GGUF + Ollama 등록
```bash
mkdir -p ~/models/solar-pro
huggingface-cli download MaziyarPanahi/solar-pro-preview-instruct-GGUF \
  --include solar-pro-preview-instruct.Q4_K_M.gguf \
  --local-dir ~/models/solar-pro

cat > /tmp/solar.modelfile <<EOF
FROM /Users/$USER/models/solar-pro/solar-pro-preview-instruct.Q4_K_M.gguf
EOF
ollama create solar-pro -f /tmp/solar.modelfile
```

## 트러블슈팅

### `huggingface-cli: command not found`
```bash
python3 -m pip install --user huggingface_hub
echo 'export PATH="$HOME/Library/Python/3.9/bin:$PATH"' >> ~/.zshrc
source ~/.zshrc
```

### `ollama: Error: model "..." not found`
```bash
./install.sh --check         # 상태 확인
./install.sh                 # 누락분만 다시
```

### MLX 변환 도중 Out of Memory
다른 앱 닫고 다시 시도. 또는 메모리 더 큰 머신에서 변환 후 디렉토리만 복사.

### A.X 다운로드 너무 느림
원본이 14GB라 어쩔 수 없음. 백그라운드로 두고 EXAONE/Solar 부터 사용.
다운로드 진행 중 멈춰도 `install.sh` 다시 실행하면 이어받음 (HF 캐시).

### `Address already in use` (포트 11434)
이미 Ollama 실행 중인 것. `pkill ollama` 후 다시.
