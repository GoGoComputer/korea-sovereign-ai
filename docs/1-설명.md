# 1. 설명 — 한국 소버린 AI 라인업

## 🇰🇷 한국 소버린 AI(Sovereign AI)란?

**소버린 AI = "주권 AI"**. 우리나라 기업·연구소가 직접 만든 AI 모델.
- 데이터가 해외 서버로 나가지 않음
- 우리말을 가장 잘 이해함
- 정부 **"독자 AI 파운데이션 모델(독파모)"** 사업 (2025~) 의 핵심 후보

이 셋업은 한국 대표 3종 모델을 **Mac M5 Pro 24GB 1대에서 모두 로컬 실행**할 수 있게 구성했습니다.

---

## 🖥️ 하드웨어 요구사항

이 패키지는 **Mac M5 Pro 24GB 통합 메모리** 기준으로 설계됐습니다.

| 항목 | 권장 | 최소 |
|---|---|---|
| 칩 | Apple Silicon (M2 Pro 이상) | Apple Silicon |
| 통합 메모리 | 24GB | 16GB (Solar 제외) |
| 디스크 여유 | 25GB+ | 6GB (--minimal) |
| OS | macOS 14+ | macOS 13+ |

> 💡 **24GB 환경 팁**: Solar(13GB)를 띄울 땐 Chrome/Slack/VS Code 같은 무거운 앱을 닫는 게 좋아요. 평소 작업용은 EXAONE/A.X 가 부담 없습니다.

> ❌ **인텔 Mac / 16GB 미만**: MLX가 동작 안 하거나 Solar가 스왑으로 매우 느려집니다. `--minimal`로 EXAONE+A.X만 설치하세요.

## 📚 모델 상세

### 【 EXAONE 4.0 1.2B 】  LG AI연구원

- **풀네임**: EXpert Ai for EveryONE 4.0
- **개발사**: LG AI연구원 (한국, 2020 설립)
- **라이선스**: EXAONE AI Model License (연구용 무료, 상업은 별도)
- **학습**: LG 자체 슈퍼컴 + 한국어/영어/스페인어 멀티링구얼
- **특징**:
  - LG 그룹 전사 AI 전략의 핵심 (가전·화학·통신 도메인 튜닝)
  - 4.0 세대는 1.2B / 7.8B / 32B 라인업 — 작은 사이즈에서도 한국어 품질 우수
  - 정부 **독파모** 사업 후보 모델
- **이 셋업 메모리**: Ollama Q4_K_M ~0.8GB / MLX 4bit ~0.7GB
- **속도**: M5 Pro MLX 4bit에서 **282 tok/s** (실측)

### 【 SKT A.X 4.0 Light 】  SK텔레콤

- **풀네임**: Adaptive eXperience 4.0 (Light 변종)
- **학습 인프라**: SKT 자체 슈퍼컴 **TITAN** (NVIDIA H100 1,000+ GPU)
- **라이선스**: **Apache 2.0** (한국 소버린 모델 중 가장 개방적, 상업 가능)
- **컨텍스트**: 32K (네이티브) / 131K (YaRN 확장)
- **특징**:
  - 한국어 토크나이저 자체 개발 → GPT-4o 대비 한국어 토큰 **33% 절약**
    (= 같은 비용으로 더 긴 한국어 처리 + 응답 빠름)
  - **KMMLU 69.2점** (한국어 종합 벤치, 동급 최상위권)
  - SKT 통신·고객 서비스 데이터로 도메인 튜닝 → 실생활 한국어 강세
  - 정부 **독파모** 사업 핵심 후보
- **이 셋업 메모리**: MLX 4bit ~4.5GB (원본 BF16 14GB)
- **속도**: M5 Pro 약 65 tok/s (실측, peak 4.16GB)

### 【 Solar Pro Preview 22B 】  Upstage

- **개발사**: Upstage (업스테이지, 한국 스타트업)
- **라이선스**: **MIT** (가장 개방적)
- **파라미터**: 22B dense — "Depth Up-Scaling(DUS)" 기법으로 효율적 확장
- **특징**:
  - 70B 급 reasoning 성능을 22B 로 달성 (Upstage 자체 벤치)
  - 한국 스타트업이 만든 글로벌 경쟁력 모델
  - 영어/한국어 모두 우수, 코드/수학 균형
  - "Preview" — 정식 Solar Pro(109B) 의 사전 공개 버전
- **이 셋업 메모리**: Ollama Q4_K_M ~13GB
- ⚠ 24GB RAM 환경: 다른 무거운 앱 닫고 실행 권장
